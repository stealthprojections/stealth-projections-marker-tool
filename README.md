# Stealth Projections Marker Tool

A desktop utility for generating BEYOND-ready laser marker CSV files from WAV tracks.

## Features
- Beat, bar, and multi-bar marker generation
- BPM override or auto-detection
- Anchor finding
- Optional phrase assistance
- Presets for fast workflow

## Usage
1. Select a WAV file
2. Enter BPM (or leave blank)
3. Choose marker preset
4. Generate CSV

## Download
Grab the latest `.exe` from the Releases tab.

## Notes
Designed for laser programmers using Pangolin BEYOND.
